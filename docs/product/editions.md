# Editions

Wayfindr is expected to have more than one distribution shape over time.

## Wayfindr Community Edition

Self-hosted, open source, AGPL-licensed core product.

Expected scope:

- live chat,
- visitor presence,
- consent-based cobrowsing,
- basic ticketing,
- audit logs,
- WordPress/Laravel/Next.js/plain JavaScript integration paths,
- Docker-based self-hosting.

## Wayfindr Cloud

Official hosted service.

Expected value:

- managed infrastructure,
- upgrades,
- backups,
- uptime,
- security maintenance,
- support,
- operational convenience.

Cloud may move faster than Community Edition.

## Future Commercial Or Enterprise Features

Some business, scale, compliance, or hosted operations features may be cloud-only or commercially licensed later. Those boundaries must be documented clearly and should not remove already-published Community Edition functionality.
